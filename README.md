# Kodeoppgave DIG

En liten frontend-applikasjon som henter ut de ulike stasjonene til Oslo Bysykkel, og viser hvor mange tilgjengelige låser og ledige sykler som finnes på dem i øyeblikket. Stasjonene vises som en liste.

Data hentes fra [Oslo Bysykkels åpne API](https://oslobysykkel.no/apne-data/sanntid).

## Teknologi

- **TypeScript** – programmeringsspråk
- **React** – rammeverk for brukergrensesnittet

## Kom i gang

Krever [Node.js](https://nodejs.org/) (versjon 18 eller nyere).

```bash
npm install
npm run dev
```

Åpne [http://localhost:5173](http://localhost:5173) i nettleseren.
